# Prompt Mejorado para el Módulo Bullet Journal

Módulo Bullet Journal - Registro Mensual
Objetivo Principal
Desarrollar un módulo de Bullet Journal digital que permita al usuario llevar un registro organizado de sus actividades diarias y hábitos mensuales, con visualización gráfica de su progreso.

Especificaciones Detalladas
1. Cabecera y Configuración Inicial
Selector de Fecha: Interfaz intuitiva para que el usuario seleccione mes y año actual

Display Principal: El mes y año seleccionado deben mostrarse en tamaño grande y destacado

Mantra/Frase Motivacional: Campo de texto dedicado para que el usuario ingrese una frase que se mostrará prominentemente cada día

2. Estructura del Registro Mensual
Sección A: Registro de Actividades Diarias
Grid Numérico: Crear una cuadrícula con números del 1 al [28, 29, 30, 31] según la cantidad de días del mes

Diseño Visual: Cada día debe tener:

Número del día claramente visible

Área de texto expandible para anotaciones

Espacio suficiente para escritura extensa (mínimo 10 líneas por día)

Organización: Diseño limpio que permita visualizar todo el mes en una sola vista

Sección B: Tracker de Hábitos
Matriz de Seguimiento: Tabla con:

Eje vertical: Lista de hábitos personalizable por el usuario

Eje horizontal: Días del mes (1-31)

Sistema de marcado: checkboxes o sistema binario (sí/no)

Personalización: El usuario debe poder agregar, editar y eliminar hábitos

3. Sistema de Visualización y Análisis
Gráfico de Progreso de Hábitos
Tipo de Gráfico: Gráfico de líneas multiple

Ejes:

Eje Y (Vertical): Porcentaje o cantidad de hábitos completados (0-100% o 0-[n] hábitos)

Eje X (Horizontal): Días del mes (1-31)

Características:

Línea individual para cada hábito o línea consolidada del progreso general

Actualización automática según los registros diarios

Puntos de datos claramente marcados

Leyenda identificando cada hábito (en caso de múltiples líneas)

4. Requisitos Técnicos y de UX
Funcionalidades Esenciales
Persistencia de Datos: Los registros deben guardarse automáticamente

Navegación entre Meses: Capacidad de ver meses anteriores y futuros

Responsive Design: Adaptable a diferentes tamaños de pantalla

Accesibilidad: Cumplir con estándares WCAG

Experiencia de Usuario
Interfaz Limpia: Diseño minimalista que priorice el contenido

Feedback Visual: Confirmaciones visuales al completar registros

Carga Rápida: Optimizado para rendimiento

Exportación: Opción para exportar datos en formatos comunes (PDF, CSV)

5. Características Adicionales Sugeridas
Personalización
Temas de color seleccionables por el usuario

Tipografías legibles y agradables

Opciones de visualización (vista compacta vs. expandida)

Análisis Avanzado
Estadísticas mensuales (porcentaje de cumplimiento, días más productivos)

Comparativas mes a mes

Recordatorios y notificaciones opcionales

Entregables Esperados
Interfaz de usuario funcional y responsive

Sistema de almacenamiento de datos robusto

Gráficos interactivos y actualizables

Documentación técnica y de usuario

Suite de pruebas (unitarias y de integración)

Criterios de Aceptación
✅ Selección fluida de mes/año

✅ Registro intuitivo de actividades y hábitos

✅ Gráfico actualizado en tiempo real

✅ Persistencia correcta de datos

✅ Experiencia móvil y desktop optimizada
